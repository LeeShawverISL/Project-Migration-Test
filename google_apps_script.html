Collaborating with AI: Google Apps Script Lesson

Learn how to create custom scripts in Google Workspace using Google Apps Script and AI.

Lesson Objectives

  • Understand the basics of Google Apps Script and its editor.
  • Collaborate with generative AI to create functional scripts.
  • Automate tasks in Google Workspace using Apps Script.

Step 1: Getting Started with Apps Script

Task: Follow these steps to create a custom script in Google Apps Script:

  1. Open a new Google Doc by typing docs.new into your browserâ€™s address bar.
  2. In the Google Doc, go to the top menu and click Extensions > Apps Script. This will open the Apps Script editor.
  3. In the editor, delete any placeholder code and paste the following script:

function onOpen() {
  const ui = DocumentApp.getUi();
  const menu = ui.createMenu('My New Menu');
  menu.addItem('Show Alert', 'showCustomAlert');
  menu.addToUi();
}

function showCustomAlert() {
  const ui = DocumentApp.getUi();
  ui.alert('This coding language can take your Docs and Sheets to the next level. How can you customize your life with Google Apps Script?');
}

Explanation of the code:

  • onOpen: Automatically adds a custom menu called "Custom Tools" when the document is opened.
  • showCustomAlert: Displays an alert with a motivational message when the menu item is clicked.

Task: Save the script, reload the Google Doc, and test the custom menu by clicking Custom Tools > Show Alert.

Step 3: Debugging and Running the Script

Task: Run your script and debug any issues. Use the following tips:

  • Check the Execution Logs for errors.
  • Ask AI for help with debugging specific errors.
  • Authorize your script if prompted (e.g., for Gmail or Calendar integration).

Enhance your project by asking follow-up questions to the AI, such as:

  • "How do I add error handling to this script?"
  • "Can you make this script run automatically on a schedule?"
  • "How can I interact with multiple Google Workspace apps in one script?"
