Create your own webpage

Follow the steps below to complete today's activity and create your first webpage with AI assistance.

Lesson Objectives

  • Learn the basics of HTML and CSS.
  • Develop effective prompting skills for generative AI.
  • Create a simple, functional webpage collaboratively with AI.

Step 1: Warm-Up

Understand how AI can assist in coding by generating HTML and CSS code.

Task: Brainstorm a topic for your webpage. Examples include your favorite hobby, a personal introduction, or a fictional product.

Step 2: Prompt Writing

Write a clear and specific prompt for the AI. THIS IS THE SKILL SET WE ARE FOCUSING ON IN THIS UNIT. Good questions get good answers. Prompting AI with garbage questions gets garbage results -- GIGO: Garbage in, garbage, out.

Example prompt:

"Write the HTML and CSS code for a webpage about my favorite hobby, photography. Include a title, a header, a paragraph about why I love it, and a background image of a camera."

Task: Share your prompt with a partner and get feedback. Revise your prompt based on their suggestions.

Step 3: Collaborate with AI

Enter your revised prompt into the AI tool. Review the generated code and ask follow-up questions to improve it.

Task: Copy the code into Replit, which is a code editor. This link shows you how. Test the webpage and make adjustments as needed.

Extension Ideas:
  • Change font color, size, background.
  • Create a box around text.
  • Create a clickable button to a different location on the web.

Step 4: Showcase and Reflect

Task: Share your webpage with peers, share ideas and celebrate your new creations.

Reflection Questions:

  • What was easy or difficult about working with AI?
  • How did feedback from your partner improve your prompt?
  • What would you like to explore next with AI and coding?
