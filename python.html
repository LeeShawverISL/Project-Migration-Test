Collaborating with AI: Build a Python Game

Follow the steps below to learn how to create a simple Python game by collaborating with AI and using MuEditor.

Lesson Objectives

  • Practice persistence and problem solving!
  • Learn how to download, install, and use Python and Pygame.
  • Learn how to download, install, and use MuEditor.
  • Collaborate with generative AI to create and refine a simple Python game.
  • Develop effective prompting and debugging skills.

Installing Python and Pygame

Task: Follow these steps to set up Python and Pygame:

These steps will likely involve challenges and snags. AI is really good and troubleshooting problems, so you can copy/paste error message and work with your AI partner to solve the problems. Don't give up.
  1. Go to the Python website.
  2. Download the latest version of Python for your operating system (Windows, Mac, or Linux).
  3. Run the installer and make sure to check the option "Add Python to PATH" before installing.
  4. Open a terminal or command prompt and verify Python is installed by typing: 
    python --version
  5. Install Pygame by typing the following command in the terminal: 
    pip install pygame
  6. Test that Pygame is installed correctly: 
    python -m pygame.examples.aliens

Step 1: Setting Up MuEditor

Task: Follow these steps to install MuEditor:

  1. Go to the MuEditor website.
  2. Download the appropriate version for your operating system (Windows, Mac, or Linux).
  3. Run the installer and complete the installation process.
  4. Open MuEditor and type the following test code:
print("Hello, MuEditor!")

Click the Run button and ensure the output appears in the console.

Step 2: Collaborating with AI

Task: Write a clear prompt to ask the AI for a Python game idea and code. THIS IS THE SKILL SET WE ARE FOCUSING ON IN THIS UNIT. Good questions get good answers. Prompting AI with garbage questions gets garbage results -- GIGO: Garbage in, garbage, out.

Example prompt:

Write Python code for a simple number guessing game. The program should:
  • Generate a random number between 1 and 100.
  • Allow the user to guess the number.
  • Tell the user if their guess is too high or too low.
  • End when the user guesses correctly and display the number of attempts.

After generating the code, copy it from the AI into MuEditor.

Step 3: Refining and Running the Game

Task: Paste the AI-generated code into MuEditor, run it, and debug as needed.

Ask the AI follow-up questions to enhance your game. For example:

  • "How can I add levels to my game?"
  • "Can you add a scoring system?"
  • "How do I improve error handling in my game?"

Experiment with adding creative features and run your updated code.

Step 4: Showcase and Reflect

Task: Pair up with a partner and test each other's games. Provide feedback:

  • What did you like about the game?
  • What features could make it more fun?

Reflection Questions:

  • What worked well when collaborating with AI?
  • What challenges did you face while debugging?
  • What would you like to explore next in Python game development?
